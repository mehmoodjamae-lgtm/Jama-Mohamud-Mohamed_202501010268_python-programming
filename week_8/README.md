# Food Delivery Receipt Generator

## Purpose

This application is used to generate a food delivery receipt. It collects customer information, calculates the subtotal, service charge, delivery charge, and displays the final receipt.

## Tech Stack

- Python 3
- Functions
- Modules
- User Input

## How to Use

1. Open terminal.
2. Go to the project folder.
3. Run:

python main.py

4. Enter:
- Customer name
- Food ordered
- Quantity
- Price per item
- Delivery (Y/N)

5. The receipt will be displayed.
